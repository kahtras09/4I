## Useful environment variables

- **DEBUG_DISPLAY** enables the Device Preview.
- **DISABLE_BKG_ANIMATION** disables the main background animation.

Use them like this:

```
flutter run --dart-define=DEBUG_DISPLAY=true,DISABLE_BKG_ANIMATION=true
```
